Simulations

Editor mode and Freestyle simulation

THIS IS A BETA FEATURE some bugs may occur.

Once you click on the Editor Mode you will be able to add more bodies to the simulation.

Clicking with the Left Button of your mouse will add a body with pre-defined mass and velocity to the clicked-location.

Clicking with the Right Button of your mouse will add a body with a dialog box to define it's mass and velocities

The Editor Mode can be used with any simulation, but the Freestyle simulation mode is perfect to use with it.

Freestyle starts a simulation on pause and no bodies, so you can add as many bodies as you want, just be careful because the more the heavier the simulation will be.

If you're gonna create a lot of bodies, turn off the trails otherwise the browser can get really slow.
